# Bookspire

책을 통해 영감을 얻는 서비스 — 한 줄로 남기는 오늘의 독서

## 실행

1. Supabase 프로젝트 생성 후 SQL Editor에서 `docs/tech-stack.md`의 스키마 실행
2. `.env.local.example`을 `.env.local`로 복사하고 Supabase URL/key 입력
3. 개발 서버 실행:

```bash
npm install
npm run dev
```
